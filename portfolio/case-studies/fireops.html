FireOps

Hybrid Security Monitoring Platform

Wazuh OpenSearch Suricata AWS CloudTrail Docker Tailscale
View on GitHub

Overview

FireOps is a hybrid security monitoring architecture that provides centralized visibility across cloud and on-premises environments by integrating enterprise security monitoring, cloud telemetry, and network detection technologies.

Key Features

  • Centralized Monitoring: Wazuh SIEM for log collection, analysis, and alerting
  • Cloud Visibility: AWS CloudTrail integration for cloud infrastructure monitoring
  • Network Detection: Suricata-based intrusion detection and network analysis
  • Data Visualization: OpenSearch dashboards for security analytics and reporting
  • Secure Access: Tailscale-based secure remote access for distributed deployment
  • Containerized: Docker-based deployment for portability and scalability

Architecture

The platform uses a hub-and-spoke architecture where Wazuh agents collect telemetry from endpoints and forward it to a central Wazuh manager. CloudTrail logs are ingested via AWS integration, while Suricata provides network-level detection. All data is indexed in OpenSearch for visualization and analysis.

Technologies

Wazuh, OpenSearch, Suricata, AWS CloudTrail, Docker, Tailscale, Linux
